LangLearnApp — Spanish/English Learning Tool

This is a Qt/C++ application designed to help users learn Spanish through sentence splitting, vocabulary practice, and AI-powered translation assistance.

Features

Split Spanish sentences into aligned Spanish/English fragments

Practice Mode (English → Spanish translation with AI validation)

Data Entry Mode (enter Spanish, get AI translation + checks)

Light/Dark themes with background images

Sentence generation from saved fragments

JSON-based memory system

Audio playback using Google TTS

OpenAI API integration (API key removed for security)

Technologies

C++

Qt 6

QNetworkAccessManager

JSON parsing

QFile / QDir for persistent storage

Custom UI with backgrounds

Notes

API key intentionally removed for security.
To run the project, add your own API key in Qt before compiling.
